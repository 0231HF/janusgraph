h2. Faunus: A Library of Graph-Based Hadoop Tools

h3. REQUIREMENTS
  * Hadoop 0.20+
  * Java 1.6+
  * Maven3

h3. USAGE

  * @mvn clean package@
  * @hadoop jar target/faunus-0.1-job.jar com.thinkaurelius.faunus.drivers.PathAlgebra jsongraph.txt out.txt@
  * @hadoop fs -getmerge out.txt ~/Desktop/out.txt@
