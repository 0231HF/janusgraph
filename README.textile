Faunus is a library for processing graphs using "Hadoop's":http://hadoop.apache.org/ "MapReduce":http://hadoop.apache.org/mapreduce/ framework. Faunus provides operators that implement a "property graph algebra":http://arxiv.org/abs/0806.2274 that can be used to process massive scale graphs.

h2. Features

* Support for various data sources
** "Titan 0.1":http://thinkaurelius.github.com/titan/
** "Hadoop File System":http://hadoop.apache.org/hdfs/ (HDFS) 1.0
** "HBase":http://hbase.apache.org/ 0.92.1
* Natively implements the "Blueprints":http://blueprints.tinkerpop.com graph API
* Open source with the liberal "Apache 2":http://www.apache.org/licenses/LICENSE-2.0.html license
