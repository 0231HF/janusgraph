Faunus is a distributed analytics engine for processing "property graphs":https://github.com/tinkerpop/blueprints/wiki/Property-Graph-Model with  "Hadoop":http://hadoop.apache.org/. A "breadth-first":http://en.wikipedia.org/wiki/Breadth-first_search version of the graph traversal language "Gremlin":http://gremlin.tinkerpop.com operates on a "vertex-centric":http://en.wikipedia.org/wiki/Adjacency_list property graph data structure. Faunus provides adaptors to the distributed graph database "Titan":http://thinkaurelius.github.com/titan/, any "Rexster":http://rexster.tinkerpop.com fronted graph database, and to text and binary graphs stored in "HDFS":http://hadoop.apache.org/hdfs. The provided Gremlin operations and Hadoop graph tools can be extended using "MapReduce":http://hadoop.apache.org/mapreduce/ and "Blueprints":http://blueprints.tinkerpop.com.
